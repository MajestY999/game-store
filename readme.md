# 🎮 Smoletik-shop

Веб-приложение для покупки цифровых товаров (игры, подписки, пополнения Steam) с функциями профиля пользователя и встроенной системой оплаты.

## ✨ Основные особенности

- 🎯 **Каталог товаров** с фильтрацией по категориям, платформам и жанрам
- 🔍 **Поиск** по названиям товаров
- 🛒 **Корзина** с управлением количеством товаров и автосохранением
- 💳 **Система оплаты** с валидацией данных карты
- 👤 **Профиль пользователя** с историей покупок
- 💬 **Чат** для общения с поддержкой/другими пользователями
- ⭐ **Рецензии** товаров с рейтингом от пользователей
- 📱 **Адаптивный дизайн** с красивыми градиентами и анимациями
- 💾 **Локальное хранилище** - все данные сохраняются в браузере

## 🛍️ Поддерживаемые категории товаров

- **Игры** для PC, PlayStation, Xbox
- **Подписки** (подписка на игры, облачные сервисы)
- **Цифровые пополнения** (Steam пополнение с кастомной суммой 50-20000 ₽)

## 🚀 Быстрый старт

### Требования

- Node.js 14+ и npm

### Установка

```bash
cd game-store/my-app
npm install
```

### Запуск в режиме разработки

```bash
npm start
```

Приложение откроется на [http://localhost:3000](http://localhost:3000)

### Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут в папке `build/`

## 💻 Технологический стек

- **Frontend Framework:** React 19.2.4 с Hooks (useState, useEffect)
- **Стилизация:** CSS (Flexbox, Grid, градиенты, анимации)
- **Хранилище данных:** localStorage + React state
- **Валидация:** Client-side validation
- **Сборка:** Create React App (Webpack, Babel)
- **Testing:** Jest + React Testing Library

## 📋 Функциональность

### 🏠 Главная страница

- Баннер с горячими предложениями
- Избранные товары
- Быстрый переход в каталог

### 📚 Каталог товаров

- Фильтрация по категориям (Игры, Подписки, Пополнения)
- Фильтрация по платформам (PC, PlayStation, Xbox)
- Фильтрация по жанрам (Action, RPG, Strategy, Adventure, Sports)
- Поиск по названию
- Сортировка по цене, популярности

### 🎮 Страница товара

- Детальная информация о товаре
- Выбор варианта (издание, срок подписки)
- Кастомный ввод суммы (для Steam пополнения)
- Рецензии пользователей с рейтингом ⭐
- Похожие товары (рекомендации)
- Добавление в корзину

### 🛒 Корзина

- Просмотр добавленных товаров
- Изменение количества
- Удаление товаров
- Расчёт итоговой стоимости
- Автосохранение в localStorage
- Рекомендации похожих товаров

### 💳 Оплата

- Модальное окно с формой оплаты
- Валидация номера карты (12 цифр)
- Проверка ФИ, даты экспирации, CVV
- Обработка ошибок с иконками и сообщениями
- Успешное завершение покупки

### 👤 Профиль пользователя

- Редактирование личных данных
- Просмотр истории покупок с деталями
- Сохранение информации в localStorage

### 💬 Чат

- Отправка и получение сообщений
- История общения
- Контакт с поддержкой
